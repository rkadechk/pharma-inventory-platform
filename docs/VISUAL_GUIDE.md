## The Script In 30 Seconds

```python
# 1. Pick random facility and drug
# 2. Generate realistic quantities
# 3. Calculate expiration dates
# 4. Assign risk levels
# 5. Write to CSV
# 6. Repeat 3,000 times for inventory
#
# 7. Repeat 182,500 times for consumption
# 8. Add seasonality (more demand in winter)
#
# 9. For 365 days, add weather/disease signals
# 10. Create demand forecasts (90 days forward)
# 11. Create replenishment orders
#
# OUTPUT: 8 CSV files with ~209,000 rows ready for:
# - Expiration Management agents
# - Multi-facility coordination engines
# - ML demand forecasting models
# - Power BI dashboards
```

## The 8 CSV Files You Create

```
facilities.csv           (5 rows)
  └─ Hospital names, locations

medications.csv          (25 rows)
  └─ Drug names, categories, shelf life

inventory.csv            (3,000 rows)     ← Most complex!
  └─ Current stock, expiration, risk level

consumption.csv          (~182,500 rows)  ← Biggest file!
  └─ Daily usage history, seasonality

transfers.csv            (500 rows)
  └─ Inter-hospital movements

external_signals.csv     (365 rows)       ← 1 per day
  └─ Weather, disease, events

demand_forecast.csv      (~22,500 rows)
  └─ 90-day predictions

replenishment_orders.csv (200 rows)
  └─ Supplier orders, status

TOTAL: ~209,000 rows
```
